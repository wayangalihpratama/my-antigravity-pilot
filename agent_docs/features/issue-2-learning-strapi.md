# Feature: Learning Strapi with Guidance Project

## Goal
The user wants to learn Strapi using the `climate-think-and-do-tank` project as a real-world reference. This "feature" tracks the structured learning and analysis process.

## User Review Required
> [!IMPORTANT]
> This is a learning-focused task. Does the user want a high-level overview or a deep technical dive into specific Strapi components (e.g., custom controllers, plugins)?

## Proposed Learning Scope
1. **Strapi Core Concepts (v5)**: High-level overview of the Headless CMS architecture, Content-Type Builder, and API Engine.
2. **Guidance Project Analysis**:
   - Extraction of "Most Used Cases" from `climate-think-and-do-tank`.
   - Understanding the Next.js + Strapi integration pattern.
   - Analysis of `config-sync` for version-controlling CMS schemas.
3. **New Stack Preparation**:
   - Design a `strapi-nextjs` stack skeleton following repo standards.

## Success Criteria
- [ ] Comprehensive understanding of Strapi v5 architecture.
- [ ] Clear map of the guidance project's backend structure.
- [ ] Ability to modify content types and API responses.
