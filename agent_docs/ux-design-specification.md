# UX Specification: Strapi-NextJS Stack

## 1. Design Vision
To provide a premium, modern, and developer-friendly starting point that feels "alive" and extremely high-quality.

## 2. Visual Standards
- **Theme**: Dark-first with high-contrast accents (similar to the repo's existing aesthetics).
- **Typography**: Inter / Outfit for a clean, technical look.
- **Aesthetics**: Glassmorphism, subtle gradients, and micro-animations for interactive elements.

## 3. Interaction Patterns
- **Next.js Frontend**:
  - Smooth page transitions using Framer Motion.
  - Responsive layouts (Mobile-first).
  - Loading states using Skeleton screens to match Strapi data fetching.
- **Strapi Admin**:
  - Standardized component icons for better editor UX.
  - Logical categorization of Content Types.

## 4. Components to Scaffold
1. **Hero**: Support for images/video (as seen in guidance project).
2. **Text Section**: Markdown-friendly rich text.
3. **Card Grid**: For collection type displays.
4. **Footer/Header**: Global single types.
